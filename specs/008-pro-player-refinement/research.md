# Research: Pro-Player Refinement — GestorPulse & Atmospheric Hero

## 1. O Novo Hero Background: "Silver Silk"
- **Problema:** Fundo preto chapado (`zinc-950`) é muito "vazio".
- **Solução:** Implementar um **Animated Mesh Gradient** em escala de cinza. Usar gradientes radiais que se movem lentamente simulando uma luz batendo em metal escovado ou seda prateada.
- **Técnica:** Variáveis CSS animadas no background do `SilverHero`.

## 2. O Caso de Estudo: GestorPulse (CRM ERP)
- **Status:** Projeto Full Stack Pro Player.
- **Destaque:** Precisa de um card diferenciado ("Featured Project") na galeria.
- **URL:** `sgc.gestorpulse.com.br`
- **Branding:** Deve seguir a estética monochrome mas com indicação de que é um sistema complexo (Full Stack).

## 3. Redesign dos Project Cards
- **Problema:** Cards atuais são muito básicos.
- **Solução:** Estilo "Glass Product". Imagem do projeto com um leve `mask-gradient`, informações técnicas (stack) em micro-tags prateadas e um botão de "Live Preview" magnético.
