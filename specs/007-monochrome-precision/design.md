# Design: Silver Precision (Monochrome 2026)

## 1. Palette: The Grayscale Mastery
A cor é o que você **não** usa.
- **Deep Black:** `#09090b` (Background).
- **Pure White:** `#ffffff` (Primary Text).
- **Silver / Chrome:** `#e4e4e7` (Highlights & Icons).
- **Mist / Zinc:** `#71717a` (Secondary Text).

## 2. Geometry: The "Consistent Edge"
Abandonamos o caos das quinas.
- **Global Radius:** `rounded-lg` (8px). 
- **Borders:** `1px solid rgba(255,255,255,0.08)` (Prata sutil).

## 3. Typography: Minimalist Sophistication
- **Headlines:** `Outfit` (ou `Satoshi`) em `font-bold`, `tracking-tight`. 
- **Body:** `Geist` em `text-zinc-400`.
- **Accents:** `JetBrains Mono` em `zinc-500` para metadados técnicos.

## 4. UI Components
- **Buttons:** Fundo branco com texto preto (Invertido) para CTAs primários. Fundo transparente com borda prata para secundários.
- **Cards:** Vidro escuro (`bg-zinc-900/30`) com desfoque de `20px`.
